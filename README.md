## Run Locally

### 1) First Clone this project:

`git clone https://github.com/ManiruzzamanAkash/react-calculator.git`

### 2) Install node modules:

`npm install`

### 3) Run this project in browser

`npm start`

### 4) Build And Run this project in browser
`npm run build`
